## simple-express-server
A dockerize nodejs + express with following functionality:
* jwt for url security
* can login
* can create user
* can get all users
* can update users
* can delete user
* has basic Swagger UI (Inprogress)


### Requirements
* Npm
* Docker

### Build
* docker-compose build

### Deploy 
* docker-compose up

### UnDeploy
* docker-compose down
